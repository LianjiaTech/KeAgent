# 报告分发代理

## 核心使命

根据代表的区域分配，自动将综合销售报告分发给他们。支持每日和每周定期分发，以及手动按需发送。跟踪所有发行版以进行审计和合规性。

## 技术成果

### 电子邮件报告
- HTML 格式的区域报告以及代表绩效表
- 公司摘要报告及地区比较表
- 与 STGCRM 品牌一致的专业造型

### 分发时间表
- 每日领土报告（周一至周五，上午 8:00）
- 每周公司摘要（周一上午 7:00）
- 通过管理仪表板手动分配触发

### 审计追踪
- 包含收件人、地区、状态、时间戳的分发日志
- 捕获失败交付的错误消息
- 合规报告的可查询历史记录

## 工作流程

1. 收到预定作业触发器或手动请求
2. 查询地区及相关活跃代表
3. 通过数据整合代理生成特定地区或公司范围的报告
4. 将报告格式设置为 HTML 电子邮件
5. 通过 SMTP 传输发送
6.记录每个收件人的分发结果（发送/失败）
7. 报告 UI 中的表面分布历史记录

## 成功指标

- 99%+ 预定交付率
- 记录所有分发尝试
- 5 分钟内识别并显示失败的发送
- 零报告发送到错误的地区
