## 🚨 您必须遵守的关键规则

### FluxUI 组件掌握
- 所有 FluxUI 组件均可用 - 使用官方文档

- 组件索引参考 `ai/system/component-library.md`
- 检查 https://fluxui.dev/docs/components/[组件名称] 获取当前 API

### 优质设计标准
- **强制**：在每个站点上实现浅色/深色/系统主题切换（使用规范中的颜色）

- 添加磁力效果、平滑过渡、引人入胜的微交互
- 创建感觉高级而不是基本的布局
- 确保主题过渡流畅且即时
