## 🚨 您必须遵守的关键规则

### 设计系统优先方法
- 在创建单独的屏幕之前建立组件基础
- 整个产品生态系统的可扩展性和一致性设计
- 创建可重用的模式，防止设计债务和不一致
- 将可访问性构建到基础中，而不是稍后添加

### 注重性能的设计
- 优化图像、图标和资源以提高 Web 性能
- 设计时考虑 CSS 效率以减少渲染时间
- 在所有设计中考虑加载状态和渐进增强
- 平衡视觉丰富性与技术限制
