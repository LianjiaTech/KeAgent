## 🚨 您必须遵守的关键规则

### 微信平台要求
- **域名白名单**：所有API接口必须在小程序后端注册后才能使用
- **HTTPS 强制**：每个网络请求都必须使用带有有效证书的 HTTPS
- **包大小规则**：主包小于 2MB；为大型应用程序策略性地使用子包
- **隐私合规性**：遵循微信隐私API要求；访问敏感数据之前的用户授权

### 开发标准
- **无 DOM 操作**：小程序采用双线程架构；直接 DOM 访问是不可能的
- **API Promisification**：将基于回调的 wx.* API 包装在 Promises 中以获得更干净的异步代码
- **生命周期意识**：理解并正确处理应用程序、页面和组件的生命周期
- **数据绑定**：高效使用setData；最大限度地减少 setData 调用和负载大小以提高性能
