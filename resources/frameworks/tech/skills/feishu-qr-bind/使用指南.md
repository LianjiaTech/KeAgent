> 参考 openakita 和官方插件实现，支持流式输出、群聊配置、手动凭证等多种功能

## 📖 快速开始

### 最简单的用法

```javascript

// 扫码绑定（默认）
const result = await feishu_qr_bind({
  account_name: 'ceo',
```

## 🎯 功能特性

| 功能 | 参数 | 说明 |
|------|------|------|
| **流式输出** | `interactive: true` | 显示交互式进度（Step 1/2/3...） |
| **手动凭证** | `app_id`, `app_secret` | 跳过扫码，直接输入凭证 |
| **群聊配置** | `group_policy` | 设置群聊策略（open/allowlist/disabled） |
| **@机器人** | `require_mention` | 群聊是否需要 @机器人 |
| **飞书发送** | `send_to_feishu` | 自动发送二维码到飞书 |

---

## 📋 参数详解

### 基础参数

| 参数 | 类型 | 默认值 | 必填 | 说明 |
|------|------|--------|------|------|
| `domain` | string | `'feishu'` | 否 | 飞书域名：`feishu`（中国）或 `lark`（国际） |
| `account_name` | string | `'ceo'` | 否 | 要绑定到哪个 agent（如 ceo、writer、social 等） |
| `timeout` | number | `600` | 否 | 授权超时时间（秒） |

### 三大核心功能参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| **`interactive`** | boolean | `true` | **流式输出**：true=显示进度，false=静默模式 |
| **`app_id`** | string | - | **手动输入 App ID**：提供后跳过扫码 |
| **`app_secret`** | string | - | **手动输入 App Secret**：提供后跳过扫码 |
| **`group_policy`** | string | `'open'` | **群聊策略**：open/allowlist/disabled |
| **`require_mention`** | boolean | `false` | **群聊@机器人**：true=需要@，false=不需要 |

### 飞书发送参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `send_to_feishu` | boolean | `false` | 是否发送二维码到飞书 |
| `user_open_id` | string | - | 接收者的飞书 open_id |

## 💡 使用场景

### 场景 1: 首次绑定（扫码）

适合第一次绑定某个 agent，需要扫码授权。

```javascript
const result = await feishu_qr_bind({
  domain: 'feishu',
  account_name: 'writer',
  send_to_feishu: true,  // 发送二维码到飞书
  user_open_id: 'ou_d6f374a3c5f7b1c0472ad5dd178e9441',
  interactive: true,
});
```

**流程**：
1. 自动生成二维码
2. 发送到飞书（如果启用）
3. 用户扫码授权
4. 自动保存配置

---

### 场景 2: 批量部署（手动凭证）

适合已有 appId/appSecret，批量绑定多个 agent。

```javascript
const agents = [
  { name: 'ceo', appId: 'cli_xxx', appSecret: 'xxx' },
  { name: 'writer', appId: 'cli_yyy', appSecret: 'yyy' },
  { name: 'social', appId: 'cli_zzz', appSecret: 'zzz' },
];

for (const agent of agents) {
  const result = await feishu_qr_bind({
    account_name: agent.name,
    app_id: agent.appId,
    app_secret: agent.appSecret,
    interactive: false,  // 静默模式
  });
  
  console.log(`${agent.name}: ${result.success ? '✅' : '❌'}`);
}
```

**优势**：
- ✅ 无需扫码
- ✅ 快速批量绑定
- ✅ 自动验证凭证

---

### 场景 3: 企业配置（群聊白名单）

适合企业内部使用，限制群聊权限。

```javascript
const result = await feishu_qr_bind({
  account_name: 'enterprise',
  group_policy: 'allowlist',   // 只允许白名单群聊
  require_mention: true,       // 需要 @机器人
  interactive: true,
});
```

**群聊策略说明**：

| 值 | 说明 | 适用场景 |
|-----|------|---------|
| `open` | ✅ 允许所有群聊 | 公开服务、社交媒体 |
| `allowlist` | 只允许配置的群聊 | 企业内部 |
| `disabled` | 禁用群聊 | 仅私聊场景 |

---

### 场景 4: 社交媒体助手（公开服务）

适合社交媒体、客服等公开服务。

```javascript
const result = await feishu_qr_bind({
  account_name: 'social',
  group_policy: 'open',        // 允许所有群聊
  require_mention: false,      // 不需要 @机器人
  send_to_feishu: true,
  user_open_id: 'ou_xxx',
  interactive: true,
});
```

---

### 场景 5: 静默模式（后台任务）

适合后台脚本、自动化任务。

```javascript
const result = await feishu_qr_bind({
  account_name: 'test',
  app_id: 'cli_xxx',
  app_secret: 'xxx',
  interactive: false,  // 不显示进度
});

// 只返回结果
if (result.success) {
  console.log('绑定成功');
}
```

## 📊 返回结果说明

### 成功返回

```javascript
{
  success: true,
  qr_path: "/tmp/openclaw/feishu_qr_2026-03-21T1014.png",
  qr_url: "https://open.feishu.cn/page/openclaw?user_code=XXX",
  device_code: "v1:eyJhbGci...",
  app_id: "cli_a934a108bab89bd3",
  app_secret: "xxxxx",
  account_name: "writer",
  send_to_feishu: true,
  sent_to_feishu: true,
  send_error: null,
  group_policy: "open",
  require_mention: false,
  message: "飞书机器人绑定成功！配置已保存到账户 [writer]"
}
```

**关键字段**：
- `success`: 是否成功
- `app_id`: 飞书应用 ID
- `app_secret`: 飞书应用密钥
- `account_name`: 绑定的 agent 名称
- `group_policy`: 群聊策略
- `message`: 结果说明

---

### 失败返回

```javascript
{
  success: false,
  error: "INVALID_CREDENTIALS",
  message: "凭证验证失败：App Secret 错误"
}
```

**常见错误码**：

| 错误码 | 说明 | 解决方案 |
|--------|------|---------|
| `INVALID_CREDENTIALS` | 凭证无效 | 检查 appId/appSecret |
| `TIMEOUT` | 授权超时 | 重新扫码 |
| `USER_CANCELLED` | 用户取消 | 重新授权 |
| `EXPIRED` | 授权码过期 | 重新生成 |
| `INIT_FAILED` | 初始化失败 | 检查网络 |

## 🔧 配置说明

### 保存位置

配置保存到 `~/.keagent/openclaw.json`

### 配置结构

```json
{
  "channels": {
    "feishu": {
      "accounts": {
        "writer": {
          "appId": "cli_a934a108bab89bd3",
          "appSecret": "xxxxx",
          "domain": "feishu",
          "dmPolicy": "open",
          "connectionMode": "websocket",
          "enabled": true,
          "groupPolicy": "open",
          "requireMention": false
        }
      }
    }
  }
}
```

### 配置项说明

| 配置项 | 说明 | 自动设置 |
|--------|------|---------|
| `appId` | 飞书应用 ID | ✅ |
| `appSecret` | 飞书应用密钥 | ✅ |
| `domain` | 飞书域名 | ✅ |
| `dmPolicy` | 私聊策略（open=无需配对） | ✅ 固定为 `open` |
| `connectionMode` | 连接模式 | ✅ 固定为 `websocket` |
| `enabled` | 是否启用 | ✅ 固定为 `true` |
| `groupPolicy` | 群聊策略 | ✅ 根据参数设置 |
| `requireMention` | 是否需要@ | ✅ 根据参数设置 |

## ❓ 常见问题

### Q1: 扫码后超时怎么办？

**A**: 重新执行绑定流程，二维码有效期为 10 分钟。

```javascript
const result = await feishu_qr_bind({
  account_name: 'ceo',
  timeout: 600,  // 增加超时时间
});
```

---

### Q2: 如何验证绑定是否成功？

**A**: 检查配置文件或发送测试消息。

```javascript
// 方法 1: 检查配置
const fs = require('fs');
const config = JSON.parse(fs.readFileSync('/home/ubuntu/.keagent/openclaw.json', 'utf-8'));
console.log(config.channels.feishu.accounts.ceo);

// 方法 2: 发送测试消息
// 在飞书中给绑定的 agent 发消息，看是否回复
```

---

### Q3: 群聊不回复怎么办？

**A**: 检查 `groupPolicy` 和 `requireMention` 配置。

```javascript
// 检查配置
const config = JSON.parse(fs.readFileSync('/home/ubuntu/.keagent/openclaw.json', 'utf-8'));
console.log(config.channels.feishu.accounts.social.groupPolicy);
console.log(config.channels.feishu.accounts.social.requireMention);

// 如果需要修改，重新绑定并设置参数
await feishu_qr_bind({
  account_name: 'social',
  group_policy: 'open',
  require_mention: false,
});
```

---

### Q4: 如何绑定多个 agent？

**A**: 使用循环批量绑定。

```javascript
const agents = ['ceo', 'writer', 'secretary', 'social'];

for (const name of agents) {
  await feishu_qr_bind({
    account_name: name,
    app_id: 'cli_xxx',
    app_secret: 'xxx',
    interactive: false,
  });
  console.log(`${name} 绑定完成`);
}
```

---

### Q5: 如何切换到飞书国际版？

**A**: 设置 `domain: 'lark'`。

```javascript
await feishu_qr_bind({
  domain: 'lark',  // 飞书国际
  account_name: 'ceo',
});
```
